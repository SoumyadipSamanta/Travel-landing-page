# TravelX - Hotel Booking Website

## 📖 Project Overview
TravelX is a modern travel and hotel booking website built using HTML, CSS, and JavaScript. 
It allows users to search destinations, view hotel packages with images, and book hotels through 
a clean, responsive, and user-friendly interface.

## ✨ Features
- Responsive navigation bar with sections
- Hero section with background image
- Hotel booking form (city, check-in, check-out, guests)
- Hotel gallery with images, descriptions, and "Book Now" buttons
- Travel packages section with styled cards
- Modern color palette and hover animations
- Responsive design for desktop and mobile

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (for interactivity)

## 🚀 How to Run
1. Download or clone the project folder.
2. Place all files (`index.html`, `style.css`, images, etc.) in the same directory.
3. Open `index.html` in any modern web browser.
4. Explore the hotel booking form, packages, and gallery.

## 📂 File Structure
- index.html → Main landing page with hotel booking section
- style.css → Stylesheet for design and layout
- images/ → Folder to store hotel and travel images
- README.txt → Project documentation
